# Agentic Design Documentation

This folder contains the Docusaurus documentation site for Agentic Design.

## Local Development

```bash
cd documentation
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

The site is deployed automatically via GitHub Actions when pushing to `main` or `master`.

**Setup:** Enable GitHub Pages in repository Settings → Pages → Source: **GitHub Actions**.

The live site will be available at: `https://vinipx.github.io/agentic-design/`
